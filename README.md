#Fcrypt
Fcrypt  --A simple file encryption/decryption tool

__Dependence__
---
Python3

PyCrypto

__Instal__
---
	sudo python3 setup.py install

__Uninstall__
---
	sudo pip3 uninstall fcrypt

__Usage__
---
	fcrypt 

	-h	Show help
	-f	Source file
	-o	Output file
	-e 	Encrypt
	-d	Decrypt
	-k	Password
	-r 	Remove the source file when done
	-m	Encryption mode.Use 'aes-ecb' or 'aes-cbc' or 'aes-cfb'
